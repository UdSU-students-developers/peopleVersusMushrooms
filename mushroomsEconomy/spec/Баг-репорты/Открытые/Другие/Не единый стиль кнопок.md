# Баг-репорт - Не единый стиль кнопок на сайте

**Статус:** Открыт  
**Приоритет:** Средний  
**Дата начала:** 21.04.2026  
**Срок окончания:**  
**Environment:** Google Chrome (последняя версия)

## Описание

На разных страницах сайта используются кнопки с разными стилями оформления. Отсутствует единый визуальный стиль для всех кнопок.

## Шаги воспроизведения

1. Открыть страницу авторизации
2. Открыть страницу создания лобби
3. Открыть страницу чата
4. Открыть страницу игры
5. Сравнить стили кнопок на всех страницах

## Фактический результат

На каждой странице кнопки имеют разный стиль.

## Ожидаемый результат

Все кнопки на всех страницах сайта выполнены в едином стиле.
