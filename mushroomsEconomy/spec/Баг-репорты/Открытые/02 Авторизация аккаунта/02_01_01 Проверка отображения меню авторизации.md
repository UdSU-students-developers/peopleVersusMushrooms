# Баг-репорт - Проверка отображения всех элементов меню авторизации

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 20.04.2026  
**Срок окончания:**  
**Environment:** Google Chrome (последняя версия)

## Описание

На странице авторизации присутствует ссылка «Забыли пароль?», которой не должно существовать.

## Предусловия

Открыта страница авторизации

## Шаги воспроизведения

1. Открыть страницу авторизации
2. Осмотреть страницу

## Фактический результат

На странице авторизации присутствует ссылка «Забыли пароль?», которой не должно быть на сайте.

## Ожидаемый результат

На странице авторизации отсутствует ссылка «Забыли пароль?». Отображаются только следующие элементы:
1. Поле «Логин»
2. Поле «Пароль»
3. Чекбокс «Запомнить меня»
4. Кнопка «Войти»
5. Ссылка «Нет аккаунта? Зарегистрироваться»
