### 5.3 damage

Нанести урон зданию или юниту по его guid. Если hp опускается до 0, сущность уничтожается.

**URL:** `POST /damage`

**Параметры**

```
{
    peopleEconomy: string; - guid экономики людей (совпадает с guid игрока)
    entityGuid: string;       - guid юнита или здания, которому наносится урон
    damage: number;           - величина урона
}
```
