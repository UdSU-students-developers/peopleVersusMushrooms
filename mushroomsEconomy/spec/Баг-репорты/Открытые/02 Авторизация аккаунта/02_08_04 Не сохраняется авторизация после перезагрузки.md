# Баг-репорт - Не сохраняется авторизация после перезагрузки страницы при активном чекбоксе «Запомнить меня»

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 21.04.2026  
**Срок окончания:**  
**Environment:** Google Chrome (последняя версия)

## Описание

При активации чекбокса «Запомнить меня», успешной авторизации и последующей перезагрузке страницы пользователь не остается авторизованным. Сессия не сохраняется, происходит перенаправление на страницу авторизации.

## Предусловия

1. Открыта страница авторизации
2. Есть зарегистрированный аккаунт

## Шаги воспроизведения

1. Ввести логин в поле «Логин»
2. Ввести пароль в поле «Пароль»
3. Поставить галочку в чекбоксе «Запомнить меня»
4. Нажать кнопку «Войти»
5. Подождать пока система перекинет пользователя в главное меню
6. Перезагрузить страницу

## Фактический результат

После перезагрузки страницы пользователь выходит из системы и перенаправляется обратно на страницу авторизации. Данные для входа не сохраняются.

## Ожидаемый результат

После перезагрузки страницы пользователь остается авторизованным. Страница авторизации не открывается.
