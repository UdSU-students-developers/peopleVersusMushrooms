## 1. Общее

### 1.1. Адрес сервера

`http://localhost:3005`

### 1.2. Используемый протокол

API полностью реализовано на http(s).

Формат возвращаемых значений `JSON`.

Все методы, если это особо не оговорено, имеют тип `POST`.

## 2. Структуры данных

### 2.1. Общий формат ответа

`T` - какие-то данные. В случае успешного ответа возвращается `result = 'ok'` и поле `data` с данными.

В случае ошибки возвращается `result = 'error'` и поле `error` с кодом и текстом ошибки

```
Answer<T>: {
    result: 'ok' | 'error';
    data?: T;
    error?: {
        code: number;
        text: string;
    };
}
```

## 3. Список запросов

| Название          | О чём                                          |
| ----------------- | ---------------------------------------------- |
| requestUnits      | Запросить создание боевых юнитов               |
| requestBuildings  | Запросить постройку зданий                     |
| damage            | Нанести урон зданию или юниту                  |
| move              | Переместить юнита на ближайшую свободную клетку |

### 3.1. Общие ошибки

* `242` - не переданы все необходимые параметры
* `4001` - экономика с таким `mushroomsEconomy` не найдена
* `9000` - неизвестная ошибка

## 4. Подробно

### 4.1. //Пример

Авторизация пользователя в системе

**Параметры**

```
{
    login: string; - логин пользователя
    hash: string; - контрольная сумма, равная hash = md5(md5(login + password) + rnd)
    rnd: number; - случайное целое число
}
```

**Успешный ответ**

```
Answer<User>
```

**Ошибки**

* `1002` - неверный пароль
* `1005` - неверный логин
* `1007` - неправильная длина логина
* `1008` - логин начинается с цифры или подчеркивания
* `1009` - недопустимые символы в логине
* `1010` - логин содержит пробелы или специальные символы

## 5. Игровые запросы

### 5.1 requestUnits

Запрос на создание боевых юнитов. Юниты не создаются мгновенно — сервис поставит их в очередь автопилоту и дёрнет ручку армии, как только будет готов.

**URL:** `POST /requestUnits`

**Параметры**

```
{
    mushroomsEconomy: string; - guid экономики грибов (совпадает с guid игрока)
    unitsType: string;        - тип запрашиваемого юнита
    unitsAmount: number;      - количество юнитов
}
```

**Успешный ответ**

```
Answer<{ success: true }>
```

**Ошибки**

* `242`  - не переданы все необходимые параметры
* `4001` - экономика с таким `mushroomsEconomy` не найдена

---

### 5.2 requestBuildings

Запрос на постройку зданий. Здания не строятся мгновенно — сервис поставит их в очередь автопилоту и выполнит, как только будет готов.

**URL:** `POST /requestBuildings`

**Параметры**

```
{
    mushroomsEconomy: string; - guid экономики грибов (совпадает с guid игрока)
    buildingsType: string;    - тип запрашиваемого здания
    buildingsAmount: number;  - количество зданий
}
```

**Успешный ответ**

```
Answer<{ success: true }>
```

**Ошибки**

* `242`  - не переданы все необходимые параметры
* `4001` - экономика с таким `mushroomsEconomy` не найдена

---

### 5.3 damage

Нанести урон зданию или юниту по его guid. Если hp опускается до 0, сущность уничтожается и попадает в очередь обновлений с флагом `destroyed: true`.

**URL:** `POST /damage`

**Параметры**

```
{
    mushroomsEconomy: string; - guid экономики грибов (совпадает с guid игрока)
    entityGuid: string;       - guid юнита или здания, которому наносится урон
    damage: number;           - величина урона
}
```

**Успешный ответ**

```
Answer<true>
```

**Ошибки**

* `242`  - не переданы все необходимые параметры
* `4002` - сущность не найдена или урон не применился

---

### 5.4 move

Переместить юнита (личинку или рабочего) на одну из ближайших свободных клеток. Клетка выбирается случайно из доступных соседних.

**URL:** `POST /move`

**Параметры**

```
{
    mushroomsEconomy: string; - guid экономики грибов (совпадает с guid игрока)
    guid: string;             - guid юнита, которого нужно переместить
}
```

**Успешный ответ**

```
Answer<true>
```

**Ошибки**

* `242`  - не переданы все необходимые параметры
* `4003` - юнит не найден или нет свободных клеток для перемещения
