## 1. Общее

### 1.1. Адрес сервера

`http://localhost:3007`

### 1.2. Протокол и формат ответа

Ручки юнитов работают по HTTP(S) и возвращают `JSON`.

Для юнитов используются `GET`-маршруты с параметрами в URL.

## 2. Структуры данных

### 2.1. Общий формат ответа

`T` — полезные данные успешного ответа.

```
Answer<T>: {
    result: 'ok' | 'error';
    data?: T;
    error?: string;
    code?: number;
}
```

### 2.2. Unit

```
{
    guid: string;          - guid юнита
    x: number;             - координата X
    y: number;             - координата Y
    hp: number;            - текущее здоровье
    speed: number;         - скорость
    targetX: number | null;
    targetY: number | null;
    type?: string;         - тип юнита (soldier | bmp | sniper | partizan)
}
```

## 3. Список запросов

### 3.1 CREATE_UNIT

`GET /unit/create/:guid/:x/:y?type=:type`

Создать юнита в армии пользователя.

**URL-параметры**

```
guid: string - guid пользователя (владельца армии)
x:    number - координата X
y:    number - координата Y
```

**Query-параметры**

```
type?: string - тип юнита, по умолчанию 'soldier' (поддерживается 'bmp')
```

**Успешный ответ**

```
Answer<Unit>
```

**Ошибки**

* `11` - ошибка авторизации
* `400` - неверные данные запроса / армия не найдена / тип не поддерживается
* `9000` - внутренняя ошибка

### 3.2 MOVE_UNIT

`GET /unit/move/:userGuid/:unitGuid/:x/:y`

Назначить целевую точку движения юниту пользователя.

**URL-параметры**

```
userGuid: string - guid пользователя (владельца армии)
unitGuid: string - guid юнита
x:        number - целевая координата X
y:        number - целевая координата Y
```

**Успешный ответ**

```
Answer<true>
```

**Ошибки**

* `11` - ошибка авторизации
* `400` - неверные данные запроса / армия или юнит не найдены
* `9000` - внутренняя ошибка

### 3.3 UNIT_TAKE_DAMAGE

`GET /unit/takeDamage/:userGuid/:unitGuid/:damage`

Нанести урон юниту в армии конкретного пользователя.

**URL-параметры**

```
userGuid: string - guid пользователя (владельца армии)
unitGuid: string - guid юнита
damage:   number - величина урона
```

**Успешный ответ**

```
Answer<{ guid: string; hp: number }>
```

**Ошибки**

* `11` - ошибка авторизации
* `400` - неверные данные запроса / армия не найдена
* `404` - юнит не найден
* `9000` - внутренняя ошибка

### 3.4 Общие ошибки

* `11` - Ошибка авторизации
* `13` - Отсутствуют обязательные параметры (name/password/token)
* `17` - Пользователь с таким именем уже существует
* `400` - Неверные данные запроса
* `404` - Не найдено
* `409` - Пользователь с таким именем уже существует
* `422` - Юнит с таким идентификатором уже существует
* `9000` - Ошибка сервера
