# Техническое задание: Интерфейс и визуализация карты (Frontend)


## 1. Общая структура приложения


Приложение состоит из нескольких последовательных экранов:


### 1.1. Авторизация и регистрация
- **Экран входа/регистрации** (первый экран при запуске).
- **Поля для регистрации:** только никнейм, пароль и подтверждение пароля.
- **Поля для входа:** никнейм и пароль.
- Валидация на уникальность ника при регистрации.
- Сохранение токена/сессии после успешного входа.


### 1.2. Лобби
- Список доступных игровых комнат.
- **Создание лобби:**
    - Название комнаты.
    - Возможность установить пароль на комнату (опционально?).
- **Присоединение к лобби:**
    - Список открытых комнат (с указанием названия, игроков, статуса).
    - Ввод пароля для закрытых комнат(?).
- **Внутри лобби (ожидание игроков):**
    - Eсть слоты для наблюдателя
    - Отображение слотов для игроков (1/4).
    - Чат внутри комнаты .
    - Кнопка "Готов" для каждого игрока.
    - Кнопка "Старт" (активна, когда все игроки готовы, только для создателя комнаты).
    - Отображение выбранных фракций игроков.


### 1.3. Игровой экран
- Переход к основной игре после старта из лобби.
- Загрузка сгенерированной карты (50x50) с сервера.


## 2. Интерфейс игрового экрана


### 2.1. Расположение элементов
- **Нижняя панель (основная):** В нижней части экрана располагается главная информационная панель.
    - **Портрет/иконка выделенного юнита/постройки** (слева в панели).
    - **Описание и параметры** выделенного объекта (здоровье, атака).
    - **Кнопки действий**(?) — располагаются справа от портрета или в центре панели.
    - **Мини-карта:** Квадратная или круглая мини-карта в углу нижней панели.(?крайне маловероятно)
- **Верхняя панель:**
    - **Ресурсы:** Отображение текущего количества нефти/грибницы и железа.
    - **Время/счет:** Возможно отображение таймера игры.(?)


### 2.2. Визуальный стиль
- **Цветовое кодирование фракций:**
    - Люди (Технологии): Синий/Стальной .
    - Грибы (Биология): Зеленый/Фиолетовый .
- **Полоски здоровья:**
    - Зеленая (свои юниты).
    - Красная (враги).
    - Полоски отображаются над юнитами/постройками.
- **Курсоры:**
    - Разные иконки курсора в зависимости от действия (стандартный, атака, перемещение, выделение рамкой)(?крайне маловероятно)

## 3. Требования к отображению карты


### 3.1. Рендеринг игрового поля
- **Вид:** Вид сверху (2D). Клетки отображаются в виде квадратов.
- **Сетка:** Карта 50x50 клеток. Размер ячейки фиксирован .
- **Ландшафт:**
    - Равнины (0) — нейтральный фон (земля/трава).
    - Вода (1) — синяя гладь.
    - Горы (2) — скалистая текстура.
- **Ресурсы:**
    - Кластеры ресурсов поверх тайлов ландшафта.
    - Визуализация типа ресурса: Нефть (темные лужи/вышки), Грибница (грибные наросты), Железо (скалы/руда).
    - Ресурсы видны сразу (туман войны отсутствует).
- **Управление камерой:**
    - Масштабирование (зум) колесиком мыши.
    - Перемещение по карте движением мыши к краю экрана или зажатой клавишей мыши.(или WASD?)


### 3.2. Рендеринг объектов
- **Постройки:**
    - Уникальная графика для каждого типа.
    - Принадлежность фракции (зеленый оттенок или флажок).
    - Полоска здоровья над постройкой.
    - Индикатор подключения к логистике (иконка трубы/канала).
- **Юниты:**
    - Отдельные спрайты для каждой фракции.
    - Четкая принадлежность (цвет).
    - **Ограничение:** 1 юнит на клетку.
- **Логистика (Трубы/Каналы):**
    - Сегменты труб/каналов отрисовываются как отдельные объекты на клетках.
    - Люди: металлические трубы с сочленениями.
    - Грибы: пульсирующие биологические каналы.


### 3.3. Зона влияния Флага
- Полупрозрачная подсветка клеток в радиусе действия Флага (синяя для людей, зеленая для грибов).(?)

## 4. Взаимодействие с картой


### 4.1. Выделение объектов
- **Левый клик:** Выделение одного юнита или постройки.
- **Рамка :** Выделение группы юнитов (только юниты, постройки не выделяются рамкой).(?)


### 4.2. Команды
- **Правый клик:**
    - По пустой клетке — перемещение.
    - По вражескому юниту/постройке — атака.
    - По дружественному юниту — следовать (?).
- **Клавиша S:** Остановка.


### 4.3. Строительство
- Выбор постройки через иконку в нижней панели.
- Подсветка клеток:
    - Зеленая — можно строить.
    - Красная — нельзя строить (с указанием причины: местность, занято, вне зоны флага).
- Левый клик для подтверждения строительства.


### 4.4. Прокладка труб/каналов
- Прокладка пути по алгоритму 
- Клик по добывающему строению → клик по хранилищу.

## 6. Критерии приемки(для тестов то есть я меня [:)])


1. **Авторизация/Регистрация:**
   - Работает создание аккаунта (ник+пароль).
   - Работает вход в существующий аккаунт.
2. **Лобби:**
   - Отображается список комнат.
   - Можно создать комнату (с паролем или без).
   - Можно присоединиться к комнате.
   - В комнате отображаются два игрока.
   - Работает кнопка "Готов".
   - Создатель может запустить игру, когда оба готовы.
3. **Игровой экран:**
   - Карта 50x50 загружается и отрисовывается.
   - Работает выделение юнитов и построек.
   - Работает отдача команд (правый клик).
   - Корректное отображение двух фракций.
   - Плавное движение юнитов .
4. **Производительность:**
   - Стабильные 30+ FPS на устройствах.
