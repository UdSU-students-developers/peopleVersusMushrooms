# Баг-репорт - Кнопка «Выйти» в лобби перекидывает в игру вместо списка лобби

**Статус:** Открыт  
**Приоритет:** Высокий  
**Дата начала:** 21.04.2026  
**Срок окончания:**  
**Environment:** Google Chrome (последняя версия)

## Описание

В созданном лобби есть кнопка «Выйти». При её нажатии пользователь перенаправляется в игру, хотя должен перенаправляться в список лобби.

## Предусловия

1. Пользователь авторизован
2. Создано или существует активное лобби
3. Пользователь зашёл в это лобби

## Шаги воспроизведения

1. Зайти в уже созданное лобби
2. Нажать на кнопку «Выйти»
3. Посмотреть, куда произошло перенаправление

## Фактический результат

При нажатии на кнопку «Выйти» в лобби пользователь перенаправляется в игру.

## Ожидаемый результат

При нажатии на кнопку «Выйти» в лобби пользователь должен выходить из лобби и перенаправляться в список доступных лобби.
