Название People vs Mushrooms/Экономика Грибов
1.Ресурсы
1.1. Железо - поглощает грибница для разрастания и тратится на строительство
1.2. Еда, которую кушают юниты и инкубатор
1.3. Энергия, которую генерирует реактор, здание потребляет 1 единицу энергии для работы, если здание не работает, то энергию оно не потребляет, роль запасов энергии за грибов выполняет жир
1.4. биотопливо, которое жрёт реактор, оно растёт само по себе 


2.Строения/структуры
2.1. Здание для добычи ресурсов(шахта\червь\аналог)
2.2. Биореактор, который ест биотопливо и генерирует электричество, поглощает биотопливо он сам по себе. Размер 1х1. 1 единицу энергии он вырабатывает за 2 единицы топлива, 1 генератор может запитать 1 строение
2.3. Электростанция производит 6 единиц энергии
2.3. Хранилища для ресурсов, не тратят энергию. Например бочки для жира и железа
2.4. Инкубатор\матка для создания личинок
2.5. Грибница генерирует топливо, появляется на карте и разрастается сама по себе, без рабочих, также служит в роли труб для ресурсов и проводов для энергии. Вокруг неё строится база, может быть сломана, может быть несколько на карте(соответственно и несколько баз)
2.6. Точка старта - инкубатор, маленький биореактор, заспанная грибница (В размере достаточная для снабжения реактора) 
2.7. Оборонительные сооружения: 
2.7.1. Стены
2.7.2. Стреляющая постройка
2.7.3. Грибные мины
2.8. Залежи ресурсов - точечные кластеры в двух соседних клетках могут быть два разных ресурсов, насыщенность ресурса от 1 до 16, шахта максимально добывает 16 единиц чего-либо:
2.8.1. Залежи железа
2.8.2. Залежи нефти( для грибов бесполезны, их используют люди)
2.9. Здания для развития личинок(определенных юнитов можно делать только после постройки этих зданий)
2.9.1. Строительный техникум - развивает личинку в строителя
2.9.2. ТЦК - развивает строителя в бойца ближнего боя
2.9.3. Стрелковый полигон - развивает строителя в бойца дальнего боя
2.9.4. Институт геопят - развивает строителя в геодезиста



3.Юниты
3.1. Геодезист бегает по карте и ищет залежи ресурсов
3.2. Строитель строит здания
3.3. Войска
3.3.1. Боец ближнего боя
3.3.2. Боец дальнего боя 
3.4. Личинка развивается в рабочего, рабочий в последующие стадии(Подробнее в 2.9)
4.Древо технологий. Игрок прокачивает древо технологий открывая для себя новых юнитов и постройки.
5.Концепция экономики(черновая)
Игрок спавнится с небольшим количеством ресурсов и строений, достаточным для поддержания жизни базы. Чтобы поддерживать и развивать базу необходимо питать биогенератор топливом. Топливо добывается само с клеток грибницы. Для создания юнитов есть инкубатор, который жрёт еду чтобы производить личинок. Личинки развиваются в другие типы юнитов, для создания каждого из которых необходимо соответствующее сооружение. Древо развития начинается с личинки, продолжается строителем и далее разветвляется на несколько веток: юнит ближнего боя, юнит дальнего боя, геодезист. Геодезист разведывает карту в поисках подземных ресурсов. После того как ресурс был найден его можно добыть с помощью шахты, но чтобы запитать шахту энергией нужно, чтобы она находилась на грибнице. Благодаря железу расширяется грибница и строятся здания. Оборонительные сооружения обороняют базу от людей. Среди них есть стены, которые можно сломать, грибные мины, которые взрываются и наносят урон по области при касании и большие грибы которые работают как стреляющие башни.
