# Техническое задание: Интерфейс и визуализация карты (Frontend)

**Игра:** Стратегия в реальном времени «Грибы против Людей»

## 1. Общая структура приложения

Приложение состоит из нескольких последовательных экранов:

### 1.1. Авторизация и регистрация
- **Экран входа/регистрации** (первый экран при запуске).
- **Поля для регистрации:** только никнейм, пароль и подтверждение пароля.
- **Поля для входа:** никнейм и пароль.
- Валидация на уникальность ника при регистрации.
- Сохранение токена/сессии после успешного входа.

### 1.2. Лобби
- Список доступных игровых комнат.
- **Создание лобби:**
    - Название комнаты.
    - Возможность установить пароль на комнату.
    - Количество игроков (4, с возможностью добавления наблюдателей).
- **Присоединение к лобби:**
    - Список открытых комнат (название, игроки (1/4), статус, наличие пароля).
    - Ввод пароля для закрытых комнат.
- **Внутри лобби (экран ожидания):**
    - Отображение 4 слотов для игроков.
    - Чат внутри комнаты.
    - Выбор фракции для каждого игрока (Люди или Грибы).
    - Кнопка "Готов" для каждого игрока. Игрок не может нажать "Готов", пока не выбрал фракцию.
    - Кнопка "Старт" (активна только для создателя комнаты, когда все игроки готовы).
    - Место для слотов наблюдателей.

### 1.3. Экран выбора стартовой позиции (Спаун)
- Новый этап после старта игры из лобби.
- **Отображение:** Карта, полностью скрытая **туманом войны**.
- **Процесс:**
    1. Обоим "игрокам" показывается одна и та же карта, но только в своем радиусе обзора они ничего не видят, кроме темноты.
    2. Игроки могут кликать на карту, чтобы выбрать точку для спауна.
    3. После подтверждения выбора:
        - **Для Людей:** На выбранной клетке появляется шахта, а под ним генерируется и отображается месторождение Нефти.
        - **Для Грибов:** На выбранной клетке появляется здание — **Грибница**.
    4. Выбор считается окончательным после подтверждения.
    5. Когда оба игрока выбрали позиции, игра начинается.

### 1.4. Игровой экран
- Переход к основной игре после выбора стартовых позиций.

## 2. Интерфейс игрового экрана

### 2.1. Расположение элементов
- **Нижняя панель (основная):**
    - **Описание и параметры** выделенного объекта (здоровье, атака, производительность).
    - **Кнопки действий** (становятся активными в зависимости от выделенного объекта)(?).
    - **Кнопки строительства**: открывают меню с доступными постройками (различаются для людей и грибов).
- **Верхняя панель:**
    - **Ресурсы (для игрока-человека):** Иконка Нефти (значение), Иконка Железа (значение).
    - **Ресурсы (для игрока-гриба):** Иконка Спор (значение), Иконка Железа (значение).

### 2.2. Визуальный стиль
- **Цветовое кодирование фракций:**
    - Люди: Синий/Стальной.
    - Грибы: Зеленый/Фиолетовый.
- **Полоски здоровья:** Над юнитами и постройками. Зеленые для своих, Красные для врагов.
- **Курсоры:**
    - Стандартный (стрелка).
    - Перемещение (при наведении на пустую доступную клетку с выбранным юнитом).
    - Атака (при наведении на врага).

## 3. Требования к отображению карты

### 3.1. Рендеринг игрового поля
- **Вид:** Вид сверху (2D). Клеточная сетка.
- **Сетка:** Карта 50x50 клеток. Размер ячейки фиксирован для четкости спрайтов.
- **Ландшафт:**
    - **Равнины (0):** Текстура земли/травы.
    - **Вода (1):** Текстура воды . Непроходима для Грибов.
    - **Горы (2):** Текстура скал . Непроходимы для Людей.
- **Туман войны:**
    - Открывается только в радиусе обзора юнитов и построек игрока.
    - Территории, которые были открыты, но потеряли обзор, переходят в состояние "раскрытой карты" , где виден ландшафт, но не объекты противника.
- **Ресурсы (Только для Людей):**
    - Месторождения **Нефти** и **Железа** изначально скрыты туманом войны.
    - Визуализация кластеров ресурсов появляется только тогда, когда юнит или постройка игрока-человека входит в радиус кластера. Кластер отображается как группа камней/кристаллов железа или темных пятен нефти на клетках равнин.
- **Управление камерой:**
    - Масштабирование (зум) колесиком мыши.
    - Перемещение по карте движением мыши к краю экрана или клавишами WASD.

### 3.2. Рендеринг объектов
- **Постройки:**
    - Визуально отличаются по фракциям: металлические у людей, органические у грибов.
    - **Буровая (Люди):** Строится только на клетке с нефтью. Соединена трубой с Хранилищем.
    - **Шахта (Общая):** Строится на железе. Требует подключения к логистике.
    - **Грибница (Грибы):** Главное здание. Пульсирует, распространяет вокруг себя грибницу по клеткам (визуальный эффект).
    - **Трубы/Каналы (Логистика):**
        - Люди: Металлические трубы, сегменты которых автоматически соединяются, образуя сеть.
        - Грибы: Пульсирующие биологические каналы, "корни", соединяющие постройки.
- **Юниты:**
    - Уникальные спрайты для каждого типа юнитов двух фракций.
    - Четкая цветовая принадлежность.
    - **Ограничение:** 1 юнит на клетку.


## 4. Взаимодействие с картой

### 4.1. Выделение объектов
- **Левый клик:** Выделение одного юнита или постройки.
- **Рамка (зажатый левый клик + движение):** Выделение группы юнитов.
- **Клик на пустой клетке:** Снимает выделение.
- В нижней панели отображается информация о выделенном объекте или группе (общее количество, среднее здоровье).

### 4.2. Команды
- **Правый клик:**
    - По пустой клетке (доступной для фракции) — перемещение выбранных юнитов.
    - По вражескому юниту/постройке — атака.
    - По дружественному юниту/постройке — пустое действие(ничего).

### 4.3. Строительство
- Выбор постройки через иконку в нижней панели (становятся доступны при наличии ресурсов).
- Подсветка клеток под курсором с учетом правил:
    - **Зеленая:** Можно строить (доступная местность для фракции, клетка пуста, есть ресурсы, в зоне строительства).
    - **Красная:** Нельзя строить (с указанием причины при наведении: "Непроходимая местность", "Требуется равнина", "Требуется нефть", "Нет ресурсов", "Вне зоны влияния").
- Левый клик для подтверждения строительства. Ресурсы списываются, появляется стройка.

### 4.4. Прокладка труб
- Интерфейс для соединения построек ( кнопка "Проложить трубу" при выделении Буровой).
- Клик по добывающему строению → клик по принимающему строению (Хранилище). Система прокладывает путь по алгоритму , избегая препятствий.

## 5. Обратная связь и информирование

### 5.2. Текстовые сообщения (в отдельной области)
- Системные сообщения для информирования игрока:
    - "Недостаточно ресурсов"
    - "Недостаточно спор/нефти"
    - "Требуется железо"
    - "Нельзя построить: клетка занята"
    - "Непроходимая местность для вашей фракции"
    - "Обнаружен кластер нефти/железа!"
    - "Юнит нанят"
    - "Здание построено"
    - "Ваша Грибница производит X спор в секунду"

## 6. Критерии приемки(для тестов то есть для меня [:)])

1.  **Авторизация/Регистрация:**
    - Работает создание аккаунта.
    - Работает вход.
2.  **Лобби:**
    - Отображается список комнат.
    - Можно создать комнату с параметрами (название, пароль, сид).
    - Можно присоединиться к комнате.
    - В комнате работает выбор фракций и кнопка "Готов". Кнопка "Старт" активируется только при готовности обоих и выборе разных фракций.
3.  **Выбор стартовой позиции:**
    - Карта загружена, но скрыта туманом войны.
    - Оба игрока могут выбрать и подтвердить свою стартовую клетку.
    - После подтверждения у людей под базой появляется нефть, у грибов — Грибница.
4.  **Игровой экран:**
    - Карта 50x50 отрисована.
    - Ресурсы игрока отображаются корректно в соответствии с его фракцией.
    - Работает выделение и отдача команд (левый/правый клик, рамка).
    - **Фракционные различия соблюдены:** Люди не могут ходить в горы, Грибы — в воду. Строительство подчиняется тем же правилам.
    - Логистика (трубы/каналы) прокладывается и отображается.
    - При разведке ресурсов (для людей) кластеры корректно проявляются.
